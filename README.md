# Competitive Shelf Intelligence

Competitive pricing, promo, and out-of-stock monitoring for specialty food brands.
Scrapes publicly available data from Walmart, Amazon, and Instacart to deliver
the competitive visibility that syndicated data (IRI/Nielsen) provides — at a
fraction of the cost. Streamlit dashboard showing price positioning, promotional
calendars, OOS events, and review trends for Cinderhaven's artisan sauce portfolio
and its competitive set.
